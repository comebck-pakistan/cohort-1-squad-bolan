# Improvements over Prompt V2

Prompt V4 significantly reduced hallucinations.

Unlike Prompt V2, the model no longer invented unrelated content such as:

- PARTY Safety Matches
- Kite Safety Matches
- Mohsin Match Factory
- Toys

The instruction to avoid guessing handwritten or missing text improved the reliability of the extraction.

---

# Remaining Limitations

Although hallucinations were eliminated, Prompt V4 is still a generic OCR prompt.

It treats the document primarily as text rather than as a structured invoice.

As a result:

- invoice relationships are incomplete
- supplier/customer grouping is weaker
- promotions are omitted
- taxation fields are omitted
- payment calculations are omitted
- business-specific invoice fields are not fully extracted

---

# Comparison with Prompt V3

Prompt V3 remains the strongest performer because it is specifically designed for distributor invoices.

Prompt V3 extracted:

- supplier
- customer
- products
- taxes
- discounts
- promotions
- totals
- payment information

using a predefined invoice schema.

---

# Overall Assessment

Prompt V4 improves extraction reliability and prevents hallucinations.

However, it is still not suitable as the production prompt because it lacks domain-specific invoice understanding.

---

# Recommendation

Use Prompt V3 for production invoice extraction.

Prompt V4 may be useful for unknown document types where a generic OCR output is required.
